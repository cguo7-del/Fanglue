# 方略 (Fanglue)

基于中国古籍智慧的策略生成工具 MVP

## 项目简介

方略是一个将中国古代典籍中的智慧应用到现代问题中的Web应用，帮助用户获得结构化、可落地的解决方案。

## 技术栈

- **前端**: Next.js (App Router) + TypeScript + Tailwind CSS + shadcn/ui
- **后端**: Next.js API Routes
- **数据库**: Supabase (PostgreSQL)
- **AI**: LLM Provider (OpenAI/Anthropic/DeepSeek)
